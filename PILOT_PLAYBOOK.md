# Playbook pilote Okado

Ce document sert de support pour lancer les premiers restaurants pilotes.

## Checklist lancement restaurant

1. Créer le compte marchand et finaliser l'onboarding.
2. Vérifier les informations restaurant : nom, adresse, téléphone, email, site, réseaux sociaux.
3. Créer une première animation à partir d'un objectif simple : avis Google, collecte email ou retour boutique.
4. Renseigner les dotations et les conditions d'utilisation de chaque lot.
5. Tester la page de jeu sur mobile.
6. Télécharger l'affiche PNG et vérifier le QR code depuis un smartphone.
7. Former l'équipe au scan/retrait du lot.
8. Faire une participation test et valider le retrait vendeur.
9. Vérifier dans Données que la participation et le statut du lot sont visibles.

## Script vendeur

- "Scannez le QR code sur l'affiche pour tenter de gagner un cadeau."
- "Si vous gagnez, vous recevrez un email avec votre QR code de retrait."
- "Présentez ce QR code au comptoir pour récupérer votre lot."
- "Le QR code ne peut être utilisé qu'une seule fois."

## FAQ retrait QR

**Le client n'a pas reçu l'email.**  
Vérifier les spams, puis utiliser le bouton de renvoi depuis la fiche lead si nécessaire.

**Le vendeur ne peut pas scanner.**  
Rechercher le client par email ou code de retrait dans la page Données, puis marquer le lot comme récupéré.

**Le QR code indique déjà retiré.**  
Ne pas remettre le lot sans vérification support. Le retrait est volontairement unique.

**Le lot n'est pas encore disponible.**  
Respecter le délai configuré dans la dotation, sauf décision exceptionnelle du responsable.

## Mentions RGPD simples

- Collecter uniquement les données nécessaires : prénom, email et consentement marketing.
- Expliquer que l'email sert à envoyer le gain et le QR code de retrait.
- Ne pas exporter les leads hors besoin commercial légitime.
- Supprimer les exports locaux obsolètes.
- Traiter toute demande de suppression ou modification d'email via le support.

## Templates pilotes recommandés

### Avis Google

- Action 1 : Avis Google.
- Dotation : petit cadeau immédiat ou remise.
- Objectif : augmenter le volume d'avis authentiques sans bloquer le jeu.

### Collecte email

- Action 1 : Collecte email.
- Dotation : gain instantané ou tirage avec petits lots.
- Objectif : créer une base opt-in locale.

### Retour boutique différé

- Action 1 : Avis Google ou réseau social.
- Dotation : cadeau disponible dans 24 h, valable 14 jours.
- Objectif : générer un retour en point de vente.
